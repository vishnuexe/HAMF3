# HAMF3
Hand-crafted Features meet Attention to catch Machine-crafted Fake in Face-Forgery

All the packages necessary with versions are given in _requirements.txt_

_HAMF3.ipynb_ is the file for testing your image

Put the image you would like to test in _test_input/test_image_ folder and open _HAM3.ipynb_ and run all cells. The output will also be saved in the folder named _output_
