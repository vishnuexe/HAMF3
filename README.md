# HAMF3
Hand-crafted Features meet Attention to catch Machine-crafted Fake in Face-Forgery

All the packages necessary with versions are given in requirements.txt

HAMF3.ipynb is the file for testing your image

Download the model from [here] ([url](https://drive.google.com/file/d/1BDM_zX1lptt-AKNd_z4G7G9S8E4NSIuF/view?usp=sharing)) and put it inside a folder named _model_

Put the image you would like to test in test_input/test_image folder and open HAM3.ipynb and run all cells. The output will be displayed within the notebook and also be saved in the folder named output.
